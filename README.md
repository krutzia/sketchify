# Sketchify

Sketchify is a clean, modern drawing application built using React, Vite, and Tailwind CSS.  
It provides a smooth drawing experience with essential tools such as brush, eraser, color selection, and canvas controls.

---

## Features

- Drawing canvas
- Brush size adjustment
- Color picker
- Eraser tool
- Clear canvas
- Save drawing
- Responsive layout

---

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

---

## Installation

1. Clone the repository:
```bash
  git clone https://github.com/krutzia/sketchify.git
cd sketchify
```

2. Install dependencies:
```bash
npm install
```

---

## Running the Project

Start the local development server:
```bash
 npm run dev
```
Then open your browser and navigate to the local server URL

---

## Build for Production

Generate an optimized build with:
```bash
 npm run build
```
